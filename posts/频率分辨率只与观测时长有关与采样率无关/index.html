频率分辨率只与观测时长有关与采样率无关

发表于 2025-12-23 分类于 雷达原理与系统 本文字数： 938 阅读时长 ≈ 7 分钟
第二章附件


图片节选自《雷达原理》（第30期）| 脉冲多普勒处理（一）——多普勒频谱采样

1. 定义模拟信号与采样过程

假设我们有一个连续时间的模拟信号 xa(t) ，其连续时间傅里叶变换（CTFT）为 Xa(f) ：

Xa(f)=∫−∞∞xa(t)e−j2πftdt

现在，我们用采样间隔 Ts 对其进行理想采样。采样率定义为 fs=1Ts 。

数学上，理想采样可以表示为模拟信号 xa(t) 与一个狄拉克梳状函数（Dirac Comb）（也称冲激串） p(t) 的乘积。

定义冲激串 p(t) ：

p(t)=∑n=−∞∞δ(t−nTs)

采样后的信号 xs(t) （理想抽样信号）表示为：

xs(t)=xa(t)⋅p(t)=xa(t)⋅∑n=−∞∞δ(t−nTs)

利用 δ 函数的筛选性质，这也是我们得到的离散序列的基础：

xs(t)=∑n=−∞∞xa(nTs)δ(t−nTs)

2. 频域推导：时域相乘等于频域卷积

根据傅里叶变换的卷积定理：时域中两个信号的乘积，对应频域中两个频谱的卷积。

xs(t)=xa(t)⋅p(t)→FXs(f)=Xa(f)∗P(f)

其中 P(f) 是冲激串 p(t) 的傅里叶变换。

关键步骤：求 P(f)

冲激串 ∑n=−∞∞δ(t−nTs) 是一个周期为 Ts 的周期函数。其傅里叶变换仍然是一个冲激串，但周期变为 fs 。

根据泊松求和公式（Poisson Summation Formula）或傅里叶级数展开推导，得：

P(f)=F{∑n=−∞∞δ(t−nTs)}=1Ts∑k=−∞∞δ(f−kfs)

注意：这里出现了系数 1Ts ，也就是采样率 fs 。

所以：

P(f)=fs∑k=−∞∞δ(f−kfs)

3. 计算卷积得到采样信号的频谱

现在我们将 Xa(f) 与 P(f) 进行卷积：

Xs(f)=Xa(f)∗[fs∑k=−∞∞δ(f−kfs)]

利用卷积的线性性质和 δ 函数的移位性质（ f(x)∗δ(x−x0)=f(x−x0) ）：

Xs(f)=fs∑k=−∞∞(Xa(f)∗δ(f−kfs))

最终得到采样信号的频谱表达式：

Xs(f)=fs∑k=−∞∞Xa(f−kfs)

4. 结论证明

观察上面的公式 Xs(f)=fs∑k=−∞∞Xa(f−kfs) ，我们可以得出严格的数学结论：

  1. 周期性：

    Xs(f) 是由原频谱 Xa(f) 在频率轴上每隔 fs 进行平移（Shift）并叠加而成的。

    如果你将 f 替换为 f+fs ，你会发现式子不变：

    Xs(f+fs)=fs∑k=−∞∞Xa(f+fs−kfs)=fs∑m=−∞∞Xa(f−mfs)=Xs(f)

    （令 m=k−1 ）

    这证明了 Xs(f) 是一个以 fs 为周期的周期函数。

  2. 对应关系：

    既然 Xs(f) 是周期函数，且周期为 fs ，那么要描述这个信号的完整频域特性，我们只需要截取一个周期的长度。

    这个“一个周期”的宽度，数学定义上就是 fs 。

  3. DFT/FFT 的离散化：

    DFT 是对上述连续周期频谱 Xs(f) 在一个周期内（通常是 [0,fs) ）进行 N 点等间距采样。

    因此，DFT 输出的 R0(m) (其中 m=0,1,...,N−1 ) 对应的物理频率范围就是这一个周期的宽度。

证毕：

由于时域采样导致了频域以 fs 为周期进行延拓，因此数字频谱的一个完整周期（即能够不重复地表示信息的最大范围）在数值上严格等于采样率 fs 。
