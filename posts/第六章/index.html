大纲

（1）杂波的平均RCS、信杂比的概念与计算（对照雷达方程）。 （2）杂波的统计特性，天线扫描引起的杂波功率谱展宽效应，杂波的谱宽和谱中 心的计算。 （3）雷达抑制杂波的主要措施。 （4）MTI/MTD抑制杂波的机理，改善因子的定义和计算方法（掌握两脉冲和三脉 冲对消的改善因子计算方法），SCV的概念及其与改善因子的关系； （5）MTI 滤波器的结构、传递函数（频率响应）、特点、设计方法，改善因子与 MTI 滤波器的频率响应，等T、变T时MTI滤波器频率响应的特点；MTI滤波器的设 计方法。 （6）MTD滤波器的结构、传递函数（频率响应）、特点、设计方法，MTD每个滤 波器的中心频率、带宽及其频率响应；MTD滤波器的设计方法。（7）盲速，出现盲速的条件，解决措施。 （8）慢速目标的检测：卡尔马斯滤波器的特点。

问题:什么是面杂波？什么是体杂波？改善因子是什么？

杂波

杂波的平均RCS计算公式

面杂波：主要指的是地杂波与海杂波，定义公式：

σc=σ0×Ac

其中 σ0 ：单位面积的散射系数（Scattering Coefficient），是一个无量纲量（或单位为 m2/m2 ），通常用分贝（dB）表示。它与擦地角、波长、极化方式及地表粗糙度有关。 Ac ：杂波单元的有效照射面积。对于照射面积，有以下的计算公式：

Ac≈R⋅θB⋅cτ2sec⁡ψ

其中 R ：雷达至杂波单元的距离； θB ：天线方位波束宽度（3dB波宽）； RθB ：分辨单元的横向宽度； cτ2sec⁡ψ ：分辨单元在地面上的径向投影长度（ c 为光速， τ 为脉宽）。
体杂波：主要指气象杂波（云、雨、雪）或干扰箔条
定义公式为：

σc=ηv×Vc

其中 ηv ：单位体积的反射率； Vc ：雷达分辨单元的体积。照射体积的计算是：

Vc≈π4(RθB)(RϕB)cτ2

其中 θB ：方位波束宽度； ϕB ：俯仰波束宽度； cτ2 ：距离分辨率。

信杂比

信杂比是指目标回波功率( Ps 或 Pr )与杂波回波功率( Pc ) 的比值。

SCR=PsPc
项目 目标 (Target) 面杂波 (Surface Clutter) 体杂波 (Volume Clutter)
RCS 模型 点目标 σt 分布式 σc=σ0Ac 分布式 σc=ηvVc
RCS 与距离关系 常数 (无关) ∝R ∝R2
接收功率 Pr ∝1/R4 ∝1/R3 ∝1/R2
信杂比 SCR - ∝1/R ∝1/R2
什么是噪声受限与杂波受限？
噪声受限：当雷达的主要干扰来源是接收机内部的热噪声（Thermal Noise）或外部的宇宙噪声，而背景杂波非常弱或不存在时，这种工作状态称为“噪声受限”。
杂波受限：当雷达的主要干扰来源是环境反射的杂波（地杂波、海杂波、雨雪等），且杂波功率远大于接收机内部噪音（ Pc≫Pn ）时，这种工作状态称为“杂波受限”。
特性 噪声受限 (Noise-Limited) 杂波受限 (Clutter-Limited)
主要干扰 接收机内部热噪声 ( Pn ) 环境反射回波 ( Pc )
典型场景 高空探测、仰视、远距离 低空探测、下视、强降雨
检测门限取决于 信噪比 (SNR) 信杂比 (SCR)
增加发射功率 有效 (SNR 提升) 无效 (SCR 不变，且可能导致接收机饱和)
改善手段 增大功率、增大天线口径、低噪放 多普勒处理 (MTI/PD)、波形设计、窄波束
随距离变化 性能随 R−4 剧烈下降 性能下降较慢 (因为杂波也随距离衰减)

杂波的统计特性

瑞利分布、对数正态分布、韦布尔分布、K分布。

杂波谱中心与谱宽

杂波在频域通常被建模为高斯函数：

S(f)=S0exp⁡[−(f−fc)22σf2]

谱中心对应杂波的平均多普勒频率，由杂波的平均径向速度决定。

  • 地杂波/固定杂波：
    由于地面、建筑物是静止的，其平均速度为0。
fc=0
  • 气象杂波/运动杂波（如云雨、鸟群、箔条）：
    它们随风漂移，具有平均径向速度 vr （即风速在雷达视线方向的分量）。
fc=fd=2vrλ

谱宽描述了杂波能量在频域的扩散程度，通常用标准差 σf 表示。它由内部运动和外部调制共同决定，这里的 σf 和 下文的 σc 指的是同一个物理量。
计算公式为所有因素的方差之和：

σf2=σwind2+σscan2+σplatform2+…

其中关键的一项是杂波内部运动引起的谱宽 ( σwind )：

  • 由物体内部的随机运动（如树叶摇动、雨滴湍流）引起。
  • 计算公式：
σwind=2σvλ 
- $\sigma_v$：杂波内部速度的均方根值（RMS velocity spread，单位 m/s）。
- $\lambda$：雷达波长。
- **规律**：波长越短（频率越高），同样的物理晃动导致的频域展宽 $\sigma_f$ 越大。

天线扫描引起的杂波功率谱展宽效应

产生原因：雷达天线在扫描过程中，波束扫过一个固定的杂波点（例如一棵树）。该杂波点被照射的时间是有限的（即“驻留时间” tot ）。时频对应关系：在时域上，回波信号受到了天线方向图（通常近似为高斯形状）的幅度调制。回波持续时间越短（脉冲串越短），在频域上其频谱就越宽（不确定性原理）。简单来说：天线转得越快，看目标一眼的时间越短，回波的频谱就被展得越宽。
天线扫描引起的谱宽 σs (Hz)：

σs≈0.265⋅frnB

其中半功率波束宽度内的脉冲数 nB 和脉冲重复频率 fr 。
总谱宽：

σc=σwind2+σs2

当雷达扫描速度很快，或者波束很窄（导致驻留时间很短）时， σs 可能成为主导因素。对MTI的影响：谱展宽会导致地杂波不仅集中在0Hz，而是扩散到邻近频率。如果MTI滤波器的凹口不够宽，这些扩散的杂波分量就会通过滤波器，导致杂波剩余，降低改善因子。因此，天线扫描速度限制了MTI性能的上限。

MTI与MTD

雷达抑制杂波的主要措施是动目标显示、动目标检测，还有一个脉冲多普勒处理。
改善因子：改善因子定义为系统输出端的信杂比（SCR）与输入端信杂比之比，并在所有可能的运动目标速度上取平均值。

I=(S/C)out(S/C)in=Ps,out/Pc,outPs,in/Pc,in

将其拆解为两部分：

I=(Ps,outPs,in)×(Pc,inPc,out)

其中左边是 Ps,outPs,in （信号增益 G¯s ）：对于在所有多普勒频率上均匀分布的目标，MTI滤波器的平均功率增益。 Pc,inPc,out （杂波衰减 CA）：系统对杂波功率的抑制能力。
SCV （杂波可见度）指的是在雷达能够使得目标在输出端“可见”（即满足检测门限要求）的前提下，输入端所允许的最大杂波功率与信号功率之比。

SCV=(CinSin)max

条件是：输出信杂比 (S/C)out 必须达到某个能够检测的阈值，这个阈值称为可见度因子 Vo 或 Dx 。

1(S/C)in=CinSin=SCV (S/C)out=Vo I=(S/C)out(S/C)in=Sout/CoutSin/Cin I=SCV⋅Vo或者SCV=IVo

分贝形式 (dB)：

SCV(dB)=I(dB)−Vo(dB)

解释说明：

  • 改善因子 ( I ) 是雷达信号处理机“挣”来的增益（把信杂比提升了多少）。
  • 可见度因子 ( Vo ) 是检测器“消耗”掉的本钱（为了保证不虚警，输出端必须保留一定的信杂比，比如 10dB）。
  • SCV 是剩下的“净利润”（实际上能抵抗多强的输入杂波）。
    例子：如果 MTI 处理器的改善因子 I=40dB （能把 SCR 提升 40dB），而为了在显示器上清楚地看到目标，需要输出信杂比 Vo=10dB 。
    那么，雷达的实际杂波中可见度为：
SCV=40dB−10dB=30dB

即：雷达能检测到比杂波弱 30dB 的目标。
虽然公式 SCV=I−Vo 看起来只要无限提高改善因子 I ，SCV 就能无限大。但在实际硬件中，SCV 的上限受限于系统不稳定性和动态范围。

标准假设：

  1. 杂波谱模型：杂波功率谱服从高斯分布。
Sc(f)=S0exp⁡(−f22σc2) 
其中 $\sigma_c$ 是杂波谱的标准差（Hz）。
  1. 谱中心：假设杂波谱中心位于零频（ fc=0 ），即地杂波。
  2. 窄带假设：假设杂波谱很窄，即 σc≪fr （脉冲重复频率）。

动目标显示（MTI）

单延迟线对消器与双延迟线对消器：

单延迟线对消器：设输入信号为 x(t) ，输出信号为 y(t) ，延迟时间为 Tr 。

y(t)=x(t)−x(t−Tr)


双延迟线对消器：

y(t)=[x(t)−x(t−Tr)]−[x(t−Tr)−x(t−2Tr)]


双脉冲的改善因子计算：

I1=2(fr2πσc)2

三脉冲的改善因子计算：

I2=2(fr2πσc)4

改善因子主要取决于比值 fr/σc （脉冲重复频率与杂波谱宽之比），详细推导过程见：第六章附件问题1.

参数 双脉冲 (单延迟线) 三脉冲 (双延迟线) 备注
时域权值 [1,−1] [1,−2,1]
信号增益 ( G¯s ) 2 6 权值平方和
频率响应近似 ∝f2 ∝f4 零频附近的特性
利用的矩 二阶矩 ( σc2 ) 四阶矩 ( 3σc4 ) 高斯分布特性
改善因子公式 2(fr2πσc)2 2(fr2πσc)4 三脉冲为四次方关系

动目标检测（MTD）

雷达发射一串相干脉冲（例如 N=8 或 N=16 个脉冲），称为一个相干处理间隔（CPI）。第 n 个脉冲为回波采样信号 x(n) ，MTD 的核心是对序列 x(n) 做离散傅里叶变换（DFT）。

|X(k)|=A⋅|sin⁡[Nπ(fdTr−k/N)]sin⁡[π(fdTr−k/N)]|

改善因子与脉冲数 N 成正比。

  • “处理”体现在哪里？（看不见的步骤）
    课件的框图中可能只看到了 FFT，但实际上 MTD 处理器在 FFT 之后隐含了关键的判决逻辑，这才是真正的“动手术”环节：
  1. 扔掉垃圾桶 (Discarding/Notching)：
    系统在读取 FFT 结果时，会有意识地忽略或压低第 0 号通道（以及其附近的通道）的数值。因为我们知道那是地杂波的老巢。这相当于在频域挖了一个坑，把杂波埋了。
  2. 区别对待 (Differential Processing)：
    • 对于第 0 通道（零速）：系统会说“这里全是杂波，除非信号强得离谱（比如比历史平均值高很多），否则我不报警”。（这就是杂波图/Clutter Map的作用）。
    • 对于第 k 通道（高速）：系统会说“这里平时应该很干净，如果突然冒出一个小尖峰，那肯定是有目标来了，报警！”。（这就是CFAR/恒虚警检测的作用）。
      中心频率 (Center Frequency)
      MTD 将整个频率范围 [0,fr) 均匀划分为 N 个通道。
  • 计算公式：
    第 k 个滤波器的中心频率 fk 为：
fk=k⋅frN,k=0,1,2,…,N−1
  • 物理意义：
    • k=0 ：中心频率为 0 Hz。这是零多普勒通道，主要包含地杂波。
    • k≠0 ：中心频率对应不同的多普勒速度。如果目标回波落入第 k 个滤波器，雷达就可以算出该目标的大致径向速度为 v=λfk2 。
      带宽 (Bandwidth)
      每个滤波器都是一个窄带滤波器。其带宽取决于观测时间的长短（即积累的脉冲数 N ）。
  • 瑞利带宽 (Rayleigh Bandwidth)：
    通常指峰值到第一个零点之间的宽度，或者分辨率带宽。
Δf≈frN=1NTr=1Tot 
（其中 $T_{ot}$ 是总的相干积累时间/驻留时间）。
特性 MTI (动目标显示) MTD (动目标检测)
数学操作 时域差分 (微分) 频域变换 (傅里叶变换/积分)
滤波器形状 宽凹口的单/双延迟线 一组窄带带通滤波器
对信号的处理 只是对消杂波，不积累信号能量 既抑制杂波，又相干积累信号能量
改善因子 有限 (20-40dB) 很高 (可达 50-60dB，取决于 N )
输出结果 只有幅度，无速度信息 幅度 + 多普勒频率(速度)

慢速目标

卡尔马斯滤波器 (Kalmus Filter) 是专门用于解决慢速目标或切向运动目标（径向速度为0）检测问题的关键组件。
卡尔马斯滤波器通常采用简单的单极点递归结构:

Mk=(1−α)⋅Mk−1+α⋅|Xk|

Mk ：第 k 次扫描后更新的平均杂波电平（即杂波图的输出）。 Mk−1 ：上一次扫描存储的平均电平。 |Xk| ：当前第 k 次扫描测量到的零通道幅度。 α ：遗忘因子（或平滑系数），通常取值很小（如 1/8 或 1/16）。
