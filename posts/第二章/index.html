第二章

雷达系统的基本组成，各部分的作用

天线与伺服控制

天线用来辐射大功率信号，接 收目标散射回波信号。为了提高雷达的探测能力，雷达基本 都是采用方向性强的天线。伺服控制系统用来控制天线的转动速度和转动方式，一般 警戒雷达需要在360°方位搜索目标，需要在方位维360°旋 转。导弹上的末制导雷达一般只需要在方位±45°范围内搜索目标，伺服电机控制天线在扇区旋转。

波形产生器

波形产生器，也称频率综合器（简称频综）。根据雷达工作的信号形式，波形产生器产生不同调制类型、 输出功率10mW到1W量级的射频激励信号，送给雷达发射机；同时给雷达接收机提供相干本振信号。由于现代雷达均为相参雷达，要求雷达在每个脉冲重复周期，发射信号频率稳定性高、初始相位恒定，对频综的性能有较高的要求。

发射机

即高功率发射部分。对射频激励信号进行放大、滤波。 警戒雷达的发射机输出功率一般在1 kW到1 MW的量级，弹载末制导雷达的发射机输出功率一般在几十瓦到百瓦的量级。

接收机

即低功率接收部分。对接收信号进行放大、混频、滤波等。雷达接收信号的功率一般在 μ W到mW的量级，特别是目标距离远，要求雷达接收机的灵敏度 较高。过去雷达接收机大多经过正交检波器，输出基带回波信号，而现代雷达接收机一般输出中频信号。

收发开关

收发开关，也称为环形器。作为单基地雷达，一般收发共用天线，在发射期 间，大功率信号经过收发开关、馈线至天线，由于发射功率大，收发开关需要保护接收机，阻止或减少大功率信号进行接收机；在接收期间，接收信号经过收发开关几乎无插损地进入接收机。

信号处理机

接收机输出信号经A/D采样、中频数字正交检波后，信号处理机完成脉冲压缩、MTI/MTD、检测、点迹凝聚等处理，提高目标回波的信噪比，同时抑制杂波和干扰。为了抑制干扰，通常需要进行副瓣对消（SLC ）、副瓣匿影（SLB）等处理；在干扰对消之前，通常需要对干扰的功率和大致类型进行判断，甚至提取干扰的特征。不同体制的雷达，信号处理的差异也较大，例如，阵列雷达需要进行数字波束形成（DBF）或自适应数字波束形成 （ADBF）等处理。

数据处理机

在帧与帧之间对点迹进行航迹关联、航迹管理、航迹滤 波，送雷达终端显示器，并生成航迹文件。

终端显示与数据传输

显示回波信号的原始视频、航迹；上传目标的点迹信息。终端显示包括A显和PPI显示。

监控设备

监控雷达各分系统的工作状态，设置雷达的工作参数。若出现故障，就给出报警或提示信息，提示操作员。

电源与供电设备

雷达发射机

发射机的任务：讲雷达提哦给你一个载波收到调制的大功率信号，经馈线和收发开关由天线辐射出去。

单级振荡式

它通常由一个高功率振荡器（最典型的是磁控管 Magnetron）直接产生大功率射频信号。
优点：结构非常简单、体积小、重量轻、造价低。
缺点：频率稳定性差：频率容易受温度、负载（天线拉偏）和电压波动影响而产生漂移；非相干性：每个脉冲起振时的初始相位是随机的（脉间随机相位），这使得它很难进行复杂的信号处理。波形单一：通常只能发射简单的矩形脉冲，无法实现复杂的调制。

主振放大式发射机

将信号产生与功率放大分开处理。其中主振器用于产生一个非常稳定、低功率的射频“种子信号”，然后放大器通过一级或多级功率放大器将信号逐级放大到所需的发射功率。
优点：频率稳定性极高：由主振器决定，受外部环境影响小。全相干性：脉冲之间的相位关系是确定的，这是实现多普勒处理（测量速度）和数字信号处理的基础。波形灵活：可以对种子信号进行复杂的调制（如线性调频 LFM），从而实现脉冲压缩，兼顾探测距离和分辨率；适用于雷达工作频率捷变。
缺点：结构复杂、体积庞大、成本昂贵，对电源要求高。

主要性能指标

  • 工作频段或波段、带宽
    雷达通常需要在一定频带内工作
  • 输出功率
    发射机送至天线输入端的功率或者在指定负载上(馈线上一定的电压驻波比) 的功率为输出功率。
    峰值功率 Pt ：发射脉冲期间 Te 射频放大的平均输出功率。
    平均功率 Pav ：脉冲重复周期 Tr 内输出功率的平均值。 Pav=ptTeTr=PtTefr 工作比： D=TeTr=Tefr
  • 总效率
    总效率=发射机的输出功率/ 输入总功率
  • 1 dB压缩点输出功率P1dB与增益
    P1dB 是这样一个功率点：在此点上，实际的输出功率比“按线性增益理论计算出的输出功率”低了 1 dB。增益是输出功率与输入功率的比值。
  • 信号的稳定性和频谱纯度
    信号的稳定性是指信号的各项参数，例如振幅、频率(或相位)、脉冲宽度及脉冲重复频率等是否随时间作不应有的变化。
    信号的频谱纯度是指信号在应有的频谱之外的寄生输出。
    带宽修正公式（为该离散分量的 单边带功率与信号功率之⽐,以d B (分⻉)计。 对于分布型的寄⽣输出 则以偏离载频若⼲赫兹的每单位频 带的单边带功率与信号功率之⽐来 衡量,以dB/Hz计）： L(fm)≈10⋅lg⁡ΔB带宽内的单边带功率信号功率−10⋅lg⁡ΔB 偏离载波频率fm(Hz),在1 Hz带宽 内⼀个相位调制边带的功率PSSB 与载波功率Ps之⽐L(fm)称为“单边 带相位噪声",简称“相位噪声“, L(fm)=10⋅lg⁡(单边带功率密度 (1 Hz 带宽内)载波功率)(dBc/Hz)

雷达接收机

接收机的基本组成


  • 收发开关 (Duplexer / T-R Switch)
    功能：这是一个“交通指挥官”。在发射期间，它把发射机的大功率能量导向天线，隔离接收机以防烧毁（保护作用）；在接收期间，它把天线接收到的微弱回波导向接收机。
  • 射频滤波器 / 预选器 (RF Filter / Pre-selector)
    功能：用于抑制外部干扰，只让雷达工作频率附近的信号通过。
  • 低噪声放大器 (LNA)
    功能：对经天线进入的微弱射频信号进行第一级放大，关键在于“低噪声”，它决定了整个系统的灵敏度。
  • 混频器 (Mixer) 与 本机振荡器 (Local Oscillator, LO)
    功能：混频器将高频的射频 (RF) 信号与本振信号相乘，下变频转换为中频 (IF) 信号。
  • 中频放大器 (IF Amplifier)
    放大：对变频后的中频信号进行多级放大。相比射频放大器，中频放大器成本更低、增益更高、稳定性更好。匹配滤波：由于中频频率固定（不随雷达载频跳变而变），很容易在这里实现匹配滤波器，以获得最大的信噪比。
  • 检波器与视频放大器 (Detector & Video Amplifier)
    检波器 (Detector)：将中频交流信号转换为代表信号包络幅度的直流/低频脉冲信号（去除载波）。视频放大器 (Video Amplifier)：将检波后的信号放大到足够电平，以便输送给后续的信号处理机或显示终端。

主要性能指标

  • 噪声系数F与噪声温度
    噪声系数 F 描述了信号经过一个系统（如放大器）后，信噪比（SNR）恶化的程度。 F=(SNR)i(SNR)o=Si/NiSo/No 其中 (SNR)i ：输入端信噪比； (SNR)o ：输出端信噪比；因为输出信噪比总是比输入差，所以 F≥1 。最终可以得到： F=1+ΔNkT0BnGa≥1 其中 k ：波尔兹曼常数 ( 1.38×10−23J/K )； T0 ：标准噪声温度 ( 290K )； Bn ：噪声带宽。具体推导可以看这里：噪声系数推导过程。
    等效噪声温度 ( Te )：假设一个理想的无噪声放大器，我们在它的输入端放一个温度为 Te 的电阻。这个电阻产生的热噪声（ kTeBn ），经过放大后，刚好等于真实放大器内部产生的噪声（ ΔN ）。 Te 的公式（证明和后面的级联放在一起）： Te=(F−1)T0 它连接了噪声系数和噪声温度。比如 F=2 (3dB)，那么 Te=290K 。
    现实中的接收机是由很多级串联的（滤波器 -> LNA -> 混频器 -> 中频放大器…），噪声系数与等效温度可以写成： Ftotal=F1+F2−1G1+F3−1G1G2+⋯+Fn−1G1G2…Gn−1$$$$Te_total=Te1+Te2G1+Te3G1G2+… 观察公式，第一级噪声 F1 是直接加在总量里的，没有任何分母去“除”它，第二级噪声 (F2−1) 被第一级增益 G1 除。如果 G1 很大（比如 20dB, 即 100倍），那么第二级、第三级的噪声贡献就会被缩小100倍甚至更多，几乎可以忽略不计。这就是为什么接收机的第一级必须是 低噪声放大器 (LNA)——它的 F 要极小，且 G 要足够大，以“压制”后续电路（如混频器）较高的噪声。证明在这里：等效噪声温度与多级系统。
    对于无源网络（如电缆、衰减器、无源滤波器），如果其损耗为 L （Loss，线性比值，不是dB），且物理温度为 T0 。
    那么它的噪声系数就等于它的损耗： F=L=1G (注：增益 G 小于1，即 1/L )
  • 灵敏度
    灵敏度表示接收机接收微弱信号的能⼒（接收机能够检测到的最小输入信号功率）。能接收的信号越微弱,则接收机的灵敏度越⾼,因⽽雷达的作⽤距离就越远。 Si,min=kT0BnFoM 识别系数或检测因子 M=(So/No) ,在工程计算中，我们更习惯用对数（dB）相加减来代替乘除。得到公式：
Si,min(dBm)=−114+10lg⁡Bn(MHz)+Fo(dB)+M(dB) 
证明：[灵敏度](/posts/灵敏度/)。

  • 动态范围
    表示接收机抗过载性能的好坏。它是当接收机不发⽣过载时允许接收机输⼊信号强度的变化范围,即最 ⼤功率与最⼩功率之⽐,

    D(dB)=10lg⁡Pi,maxPi,min=20lg⁡Ui,maxUi,min

    1dB增益压缩点动态范围：指当接收机的输出功率大到产生 1 dB 增益压缩（即实际增益比理论线性增益下降了 1dB）时，此时的输入信号功率 ( Pi−1 ) 与最小可检测信号功率 ( Pi,min ) 之比。

    DR−1=Pi−1Pi,min

    根据增益 G=Pout/Pin ：

    DR−1=Po−1Pi,minG

接收机三种控制方式

自动增益控制AGC：AGC 是一种负反馈闭环控制系统。它检测接收机输出端的信号平均电平，产生一个反馈电压去调节中频放大器（或射频放大器）的增益。当输入信号变强时，AGC 降低增益；信号变弱时，增加增益，从而使输出信号的幅度保持相对稳定。
瞬时自动增益控制IAGC：IAGC 的原理与 AGC 类似，但其核心区别在于时间常数极小（反应极快）。它能在一个脉冲的持续时间内迅速动作。当遇到强干扰或强杂波时，增益瞬间降低；干扰一过去，增益瞬间恢复。
时间灵敏度控制STC：- STC 是一种开环控制。雷达发射脉冲的瞬间（ t=0 ），接收机增益被压到最低，随着时间推移（距离增加），接收机增益按预设的曲线（通常是 R4 曲线）逐渐恢复到最大值。

匹配滤波器

匹配滤波器并不是为了完美恢复信号的形状，而是为了让输出端的信噪比 (SNR) 达到最大。 附件中明确定义：当输入端出现信号与加性白噪声时，使其输出信噪比最大的滤波器，就是一个与输入信号相匹配的最佳滤波器。
频域特性 ( H(ω) )：

H(ω)=S∗(ω)e−jωt0

时域特性 ( h(t) )：

h(t)=s∗(t0−t)

输出信噪比：

SNRo=2EN0

其中 E ：信号的能量。 N0 ：噪声功率谱密度。
在现实电路中，构造一个完美的数学上的“匹配滤波器”（比如产生一个 sinc 函数形状的滤波器频谱）往往很难或成本很高。因此，工程师通常使用常见的模拟滤波器（如矩形滤波器、高斯滤波器、RLC电路等）来近似匹配滤波器。这就是准匹配滤波器。我们需要调整滤波器的带宽 ( B )，使其与信号的脉宽 ( τ ) 达到一个最佳的配合比例，这就是最佳时宽带宽积 ( Bτ )。

天线

天线的主要分类

主要性能指标

  • 天线效率
    天线效率衡量的是：天线到底把多少“吃进去”的能量转化成了无线电波发射出去，而不是变成了热量浪费掉。天线效率 ηA 定义为辐射功率与输入功率的比值：
ηA=PΣPin=PΣPΣ+Ploss 
其中$P_{\Sigma}$：天线辐射的功率。$P_{loss}$：损耗功率。主要来源于天线的**铜耗**（金属电阻）、**介质损耗**（绝缘材料发热）、加载元件损耗以及接地损耗.发生器输出 ($P_G$)：发射机产生的功率。进入天线 ($P_{in}$)：由于存在反射（由反射系数 $\Gamma$ 决定），真正进入天线的功率是 $P_{in} = (1 - |\Gamma|^2) P_G$。天线辐射 ($P_{\Sigma}$)：天线成功发射到空间的功率。损耗 ($P_{loss}$)：天线自身变成热量的功率。
  • 输入阻抗
    输入阻抗衡量的是：从馈电点往天线看进去，它相当于一个多大的电阻和电容/电感的组合。.取决于：天线本身的结构与尺寸;工作频率;临近天线周围物体的影响。
  • 天线方向图
    天线方向图是天线辐射场在空间中的分布图像。它直观地告诉我们：天线往哪个方向发射的能量强，往哪个方向发射的弱。天线辐射的电场强度 E : E(r,θ,φ)=60IAr⏟幅度衰减因子⋅f(θ,φ)⏟方向函数⋅e−jkr⏟相位因子 其中 60IAr （幅度与距离）： IA 是天线上的电流。 1/r 表明电场强度随距离增加而衰减（球面波扩散特性）。系数 60 源自自由空间波阻抗 120π≈377Ω 的推导（针对偶极子等线天线）。
    e−jkr （相位）：代表电磁波随距离 r 的相位滞后（ k 为波数）。它只影响波动的相位，不影响幅度大小。
    f(θ,φ) （方向函数）：这是最关键的一项。它表明电场的大小与距离 r 无关，只与观察的角度（方位角 θ 和俯仰角 φ ）有关。物理意义：这就是天线“形状”的数学描述。
    归一化方向图：为了方便绘图和比较（比如把最大值定为 1 或 0 dB），我们使用归一化方向函数 F(θ,φ) 。定义公式： F(θ,φ)=|E(θ,φ)||Emax|=f(θ,φ)fmax 功率方向图：电场方向图描述的是电压（V/m），而我们往往更关心能量（W/m²）。公式： Φ(θ,φ)=F2(θ,φ) 因为功率密度与电场强度的平方成正比 ( S∝E2 )。
  • 增益
    增益 G(θ,φ) 的标准数学表达式是： G(θ,φ)=ηA⋅D(θ,φ) 或者在最大辐射方向上：
G=ηA⋅Dmax 
**$D$ (方向性系数)**：描述的是**纯粹的空间聚焦能力**。

有效孔径、增益、半功率波束宽度之间的关系

  • 天线有效孔径与增益的关系 Gt=4πAeλ2 其中 Gt (增益)：表示天线把能量汇聚的能力。 Ae (有效孔径)：天线捕获电磁波的有效面积。它通常小于天线的物理面积 Aphy （因为有损耗），关系为 Ae=ηAphy 。
    根据衍射理论，一个面积为 Ae 的口径，其能实现的极致增益就是 4πAe/λ2 。 结论：天线面积 Ae 越大，或者波长 λ 越短（频率越高），增益 Gt 就越高。
  • 半功率波束宽度 θ3dB=λD(单位：rad) 其中** D ：天线的口径尺寸（直径或长度）。 θ3dB **：波束宽度（主瓣宽度）。
  • 增益与波束宽度的关系 Gt=K4πθαθβ(单位：弧度) 也可以写成： Gt≈26000θαθβ(单位：度) θα,θβ 是方位角和俯仰角的波束宽度。
    尺寸决定波束：天线做得越大 ( D↑ ) → 波束就越窄 ( θ↓ )。(参考公式： θ=λ/D )
    波束决定增益：波束越窄 ( θ↓ ) → 能量越集中 → 增益越高 ( G↑ )。(参考公式： G=26000/(θ⋅ϕ) )
    最终结论：天线面积 ( Ae ) 越大，增益 ( G ) 越高，波束 ( θ ) 越窄。
