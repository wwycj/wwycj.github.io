雷达辐射功率密度

整个过程可以分为三个部分：

第一阶段：能量到达目标（入射功率密度 S1 ）

这是雷达波发出去的过程。

  1. 各向同性扩散（理想情况）：
    假设雷达是一个全向点源，发射功率为 Pt 。能量会像气球一样向四周扩散成一个球面。
    在距离 R1 处，球体的表面积是 4πR12 。
    此时的功率密度（单位面积上的功率）是：

    S1′=Pt4πR12

  2. 天线增益修正（实际情况）：
    实际雷达使用定向天线，增益为 Gt 。这意味着能量被汇聚了，比各向同性源强了 Gt 倍。
    由此得到的第一个核心公式：

    S1=S1′Gt=PtGt4πR12(W/m2)
    • S1 ：目标位置处的入射功率密度。
    • 物理意义：每一平方米的目标表面上，接收到了多少雷达发射的能量。

第二阶段：目标截获并反射能量（二次辐射功率 P2 ）

当电磁波打在目标（如飞机）上时，目标会截获一部分能量并将其反射（散射）出去。

  1. 雷达散射截面积 ( σ )：
    目标能截获多少能量，取决于目标的“雷达散射截面积”（RCS，即 σ ）。这就好比目标用来“接雨水”的盆子有多大。
  2. 反射总功率 ( P2 )：
    入射密度 S1 乘以 有效面积 σ ，就是目标反射回去的总功率。
    对应图片中的中间公式： P2=S1σ=PtGtσ4πR12(W)
    • P2 ：目标作为一个“二次辐射源”，向外辐射的总功率。

第三阶段：能量返回雷达（回波功率密度 S2 ）

目标反射回来的能量， P2 ，再次向四周扩散（通常假设目标反射是各向同性的）。

  1. 二次球面扩散：
    能量从目标飞回接收天线，距离为 R2 。
    能量再次扩散到一个新的球面上，表面积为 4πR22 。
  2. 接收天线处的密度 ( S2 )：
    将反射总功率 P2 除以新的球面积。
    对应图片中的最终密度公式： S2=P24πR22 将 P2 的表达式代入，得到完整的展开式： S2=PtGtσ(4π)2R12R22(W/m2)
    • S2 ：雷达接收天线位置处的回波功率密度。
    • 物理意义：经过了双程传播（一来一回）和目标反射后，最后回到雷达天线口面的能量微弱程度。

基本雷达方程

从上面的公式得到接收天线处的密度，收天线用它的有效孔径面积 ( Ar ) 来“接住”这些回波。
最终接收功率 Pr ：

Pr=S2⋅Ar=PtGtσAr(4π)2R4$$我们将接收功率$Pr$设定为接收机能检测到的∗∗最小信号$Smin$∗∗，此时的距离$R$即为∗∗最大作用距离$Rmax$∗∗。$G=4πAλ2$，发射增益与接收增益相同：$$Rmax=[PtG2λ2σ(4π)3Smin]1/4

将最小可检测信号 Smin 展开为噪声和信噪比的要求： Smin=kT0BF(S/N)min 。

Rmax=[PtG2λ2σ(4π)3kT0BF(S/N)min]1/4

由以上的公式可以得到： Rmax∝Pt1/4 ； Rmax∝A1/2 或 G1/2 ； Rmax∝σ1/4 ； R∝1/λ 。

RCS

RCS 是用来描述目标后向散射特性的物理量。当雷达波照射到目标上时，目标会将一部分能量向四面八方散射。RCS 专门衡量反射回雷达方向（后向）的那部分能量的强弱。
RCS ( σ ) 的定义式为：

σ=P2S1

其中 σ ：目标的雷达散射截面积（单位通常为 m2 或 dBsm）。 P2 ：目标散射的总功率。 S1 ：照射到目标处的功率密度。
“RCS 的大小与目标散射总功率和照射的功率密度没有关系，RCS 是目标自身的一种固有属性。 它只取决于目标本身的特性，例如：目标的形状、目标的材料、雷达波的频率、观察视角等

电波传播过程的影响因素

大气层

  1. 传输损耗（衰减）：能量被大气吸收了。
  2. 大气衰落：信号强度的随机波动。
    衰减系数 ( δ )：单位是 dB/km。
    频率敏感性：
  • f<1 GHz ：大气衰减可忽略不计。
  • f>3 GHz ：必须考虑衰减。
  • f>10 GHz ：频率越高，衰减越严重。

三分之四地球模型

大气对电波的折射作用等效于增加了雷达的视线距离。一种处理折射的通用方法是用等效地球代替实际地球，假定等效地球半径 re=kr0 ， r0 为实际地球半径(6371km)

波瓣分离现象

当雷达照射低空目标时，电磁波会通过两条路径到达目标：直接路径和地面反射路径。这两路信号在空间中叠加，产生干涉。

  • 同相叠加：信号增强。
  • 反相抵消：信号减弱甚至消失。
    这会导致雷达原本平滑的波束在仰角维上分裂成许多细碎的波瓣（像手指一样张开），这就是波瓣分裂（由于反射面引起仰角维的波瓣分裂现象）。在某些仰角上可能探测不到目标。

云雨下的雷达方程

在恶劣天气（云、雨、雾）下，大气衰减系数 δ 会显著增加，必须修正理想的雷达方程。
定义损耗 ( La )：设大气的单程衰减系数为 δ (dB/km)，目标距离为 R 。双程总损耗（分贝值）： 2R×δ 。转换为线性损耗因子：

La=100.2δR

将损耗因子 La 加入雷达方程的分母中：

Rmax′=[PtG2λ2σ(4π)3kT0FnD0CBLLa]1/4

这里的 Rmax′ 是考虑衰减后的最大作用距离。
由于损耗 La 本身就取决于距离 R ，所以方程变成了：

Rmax′=Rmax⋅10−0.05δRmax′

几种雷达方程

双基地雷达方程

普通的雷达是单基地（Monostatic）的，即发射和接收共用一个天线。而双基地雷达的发射机和接收机分别位于两个不同的地点，且两者之间的距离（基线距离 L ）与目标距离相当。双基地雷达特别适合探测隐身飞机（因为隐身主要针对后向反射设计）。

搜索雷达方程

搜索雷达的任务不是盯着一个点看，而是在规定的时间 ts 内，扫描一个规定的立体角 Ω （覆盖范围），并发现目标。对于搜索雷达来说，频率的选择不影响能量探测距离（只影响精度和大气衰减）。如果发射功率受限，就必须增大天线面积来补偿，反之亦然。

高低重频雷达方程

  • 低重频
    脉冲重复频率低，脉冲间隔大，占空比低。
    距离不模糊，速度模糊。
  • 高重频
    脉冲发得极快，占空比很高（甚至接近 50%），每个脉冲能量小，但单位时间内脉冲数量巨大。
    距离模糊，速度不模糊。
    为什么低重频距离不模糊、速度模糊，而高重频相反：第三章附件第一个问题。

干扰下的雷达方程

估雷达性能的核心指标从 信噪比 (SNR) 变成了 信干比 (SJR - Signal to Jamming Ratio)。我们需要分别计算目标回波功率 ( S ) 和 雷达接收到的干扰功率 ( J )，然后相除。
