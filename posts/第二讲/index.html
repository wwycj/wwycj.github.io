第二讲

知识点

  1. 子空间和性质、分类、零子空间
    设 V1 是线性空间 V 的非空子集。如果 V1 对 V 中已有的加法和数乘运算封闭（即 x,y∈V1⇒x+y∈V1 且 k∈K,x∈V1⇒kx∈V1 ），则称 V1 为 V 的线性子空间。
    性质：子空间与原空间共享零元素；元素的负元素仍在子空间中；子空间的维数不超过原空间的维数 ( dim(V1)≤dim(V) )
    分类：平凡子空间： 零子空间 {0} 和空间 V 本身；非平凡子空间： 除了上述两个以外的子空间。
    零子空间不含任何线性无关的向量；零子空间没有基、规定维数为零。
  2. 生成子空间
    由一组向量 x1,...,xm 的所有线性组合构成的集合，记为 L(x1,...,xm) 或 Span(x1,...,xm) ，它也是一个子空间。
  3. 值域与核空间
    值域：矩阵 A 的列向量生成的子空间，记为 R(A) 。 A 的秩等于其值域的维数。 R(A)=dim(R(A))
    核空间（零空间）：齐次方程组 Ax=0 的解集，记为 N(A) 。核空间的维数称为零度 ( n(A) ) 。 n(A)=dimN(A)
  4. 基扩定理
    m 维子空间的基可以扩充为整个 n 维空间的基。换句话说，可以在子空间外找到 n−m 个元素与原来的基一起构成全空间的基。
  5. 子空间交与和
    交 ( V1∩V2 )： 同时属于两个子空间的元素集合。和 ( V1+V2 )： 由两个子空间的元素相加构成的集合 {x+y|x∈V1,y∈V2} 。
    定理： 子空间的交与和仍然是线性子空间 。
    维数公式：
    dim(V1+V2)+dim(V1∩V2)=dim(V1)+dim(V2)
  6. 子空间直和
    定义： 若和空间 V1+V2 中的任一元素 x 只能唯一地表示为 V1 中的元素与 V2 中的元素之和（即分解唯一），则称其为直和，记为 V1⊕V2 。
    等价条件：
    1. V1+V2 是直和。
    2. V1∩V2={0} (交集只有零元素)。
    3. dim(V1+V2)=dim(V1)+dim(V2) (维数直接相加)。
    4. V1 的基与 V2 的基合起来构成 V1+V2 的基。
  7. 线性变换
    设 T 是从线性空间 V 到自身的映射。如果 T 满足线性性质： T(kx+ly)=k(Tx)+l(Ty)(x,y∈V;k,l∈K) ，则称 T 为线性变换 。
    性质：把零元素变为零元素；负元素的象为原来元素的象的负元素；把线性相关的元素组仍变为线性相关的元素组。
