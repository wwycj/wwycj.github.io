大纲

（1）相控阵雷达的基本原理、基本组成、特点，优缺点及发展趋势。 （2）相控阵天线波束的扫描方法，波束指向及其（移相器）相位控制码的计算， 在不同方向的波束宽度计算；在一定角度范围内电扫时不出现栅瓣对天线间隔的要求。 （3）数字波束形成的原理和方法，不同指向DBF加权矢量的计算。 （4）ADBF的作用、准则及其权值的计算。 （5）数字阵列雷达的基本概念。

相控阵雷达

波束

基于您上传的图片以及提供的课件《第10章 相控阵雷达与数字阵列雷达.pdf》，我为您详细梳理了相控阵天线波束扫描的数学原理、相位控制码计算、波束宽度变化规律以及栅瓣抑制条件。

以下是结合理论推导与工程实现的详细说明：

1. 相控阵波束扫描方法与方向图推导

相控阵的核心原理是**“波的干涉”**。通过控制阵列中各个单元的馈电相位，改变阵列口径上的相位分布，从而改变等相位面（波前）的方向。

(1) 阵列因子推导

假设有一个由 N 个全向天线单元组成的等距线阵，单元间距为 d 。若第 n 个单元（ n=0,1,...,N−1 ）的电流幅度为 In ，且相邻单元间的相位差为 ΔφB 。
在远场方向 θ 处，各单元信号到达观察点的波程差导致的相位差为 kdsin⁡θ （其中 k=2πλ ）。
总的合成场强 E(θ) 是各单元场强的矢量叠加：

E(θ)=∑n=0N−1Inej(nkdsin⁡θ−nΔφB)

若各单元电流幅度相等 ( In=I0 )，利用等比数列求和公式，取模归一化后得到阵列方向图函数（即您图片右上角的公式）：

|F(θ)|=|sin⁡[N2(kdsin⁡θ−ΔφB)]Nsin⁡[12(kdsin⁡θ−ΔφB)]|

将 k=2πλ 代入，即得到图片中的形式：

|F(θ)|=1N|sin⁡[πNdλ(sin⁡θ−sin⁡θ0)]sin⁡[πdλ(sin⁡θ−sin⁡θ0)]|

(注：图片公式中的 sin⁡θ0 即对应波束指向所需的相位项)

(2) 波束扫描方法

要使波束指向空间中的 θB （即 θ0 ）方向，必须使该方向上的信号同相叠加，即方向图函数的分母为0（利用极限法则此时函数值为最大值1）。
条件为： kdsin⁡θB−ΔφB=0
由此导出波束指向与相位差的关系 3：

ΔφB=2πdλsin⁡θB

2. 波束指向与移相器相位控制码计算

在工程实现中，相位不是连续可变的，而是通过数字移相器（如4位、5位、6位）来实现的。这就涉及到“相位控制码”的计算。

(1) 相位计算过程

假设雷达波束需要指向 θB ：

  1. 计算理想相位差： 根据公式 Δφideal=2πdλsin⁡θB 。
  2. 计算第 n 个阵元的理想相位： φn=n×Δφideal(mod2π) （对 2π 取模，因为相位是周期的）。

(2) 相位量化与控制码

假设使用 M 位的数字移相器（例如课件中提到的5位移相器 4）：

  • 最小量化步长（LSB）： Δstep=360∘2M 。
    • 例如 5位移相器， Δstep=36032=11.25∘ 。
  • 计算控制码（Control Code）：
Cn=round(φnΔstep) 
其中 $C_n$ 是一个 $0$ 到 $2^M-1$ 之间的整数，这就是送给移相器的二进制控制码。
  • 实际相位： φactual=Cn×Δstep 。
    课件示例验证：
    若 N=8,d=λ/2 ，要求波束指向 θB=15∘ 。
  • 理想相位差 ΔφB=πsin⁡(15∘)≈46.58∘ 。
  • 对于第1号阵元 ( n=1 )，理想相位 46.58∘ 。
  • 使用5位移相器（步长 11.25∘ ）： 46.58/11.25=4.14 。
  • 四舍五入取整得 4，对应的二进制码为 00100。
  • 实际相位为 4×11.25∘=45∘ 。

3. 波束宽度的计算（扫描展宽效应）

波束宽度通常指半功率波束宽度（HPBW, θ0.5 或 θ3dB ）。

(1) 法线方向 ( θB=0∘ )

对于均匀分布的长线阵，半功率点出现在 N2kdsin⁡θ≈1.39 处。
由此推导出的法线波束宽度为：

θ0.5≈0.886λNd (弧度)

(2) 扫描方向 ( θB≠0∘ )

当波束扫描到 θB 方向时，天线的有效孔径（投影孔径）由 L=Nd 变为 Leff=Ndcos⁡θB 。
孔径减小导致波束变宽，具体的近似公式为 6：

θ0.5(θB)≈0.886λNdcos⁡θB (弧度)

或者转换为角度制（如您图片右下角所示）：

θ0.5(θB)≈50.8λNdcos⁡θB (度)

结论： 随着扫描角度 θB 的增大（偏离法线）， cos⁡θB 减小，波束宽度变宽。例如当扫描到 60∘ 时，波束宽度是法线方向的2倍（因为 cos⁡60∘=0.5 ）。

4. 消除栅瓣的条件与推导

栅瓣（Grating Lobes）是指除了主波束指向外，在实空间（Visible Space）内出现的其他具有最大辐射强度的波束。这是空间采样的混叠现象。

(1) 栅瓣产生的数学条件

由阵列因子公式可知，当分母为0时也会出现极大值。即：

πdλ(sin⁡θ−sin⁡θB)=mπ,m=±1,±2,...

整理得栅瓣出现的角度 θm 满足 ：

sin⁡θm=sin⁡θB+mλd

(2) 抑制栅瓣的推导

为了不出现栅瓣，也就是要求除 m=0 (主瓣) 以外的所有 m≠0 对应的 sin⁡θm 都在实空间之外，即 |sin⁡θm|>1 。
最危险的情况是主波束扫描到最大角度 θmax 时，最靠近实空间的栅瓣（ m=−1 ）进入实空间。
即我们要求：

|sin⁡θmax−λd|>1

因为 sin⁡θmax>0 且 λ/d>0 ，为了使栅瓣不出现（即 sin⁡θm<−1 ），我们需要：

sin⁡θmax−λd<−1 λd>1+sin⁡θmax

由此得到阵元间距的约束条件（即您图片中间右侧的公式）：

d<λ1+|sin⁡θmax|

特例：

  • 若要求全空域不出现栅瓣（ θmax=90∘ ），则 d<λ1+1=λ2 。这就是经典的半波长间隔要求的由来。
  • 若只扫描到 ±30∘ ( sin⁡30∘=0.5 )，则 d<λ1.5=0.67λ 即可。

数字波束形成

DBF 技术是针对阵列天线，利用数字信号处理在期望的方向形成接收波束的技术 。

DBF 的实现方法

DBF 的过程类似于时间滤波，是对采样数据进行加权求和的过程：

  • (1) 信号导向矢量 (Steering Vector)

对于来自方向 θi 的信号，其在阵列上的导向矢量 a(θi) 描述了各阵元间的相位关系：

a(θi)=[1,ejkdsin⁡θi,...,ejk(N−1)dsin⁡θi]T

其中 k=2π/λ 。

  • (2) 目标方向的权矢量
    若要使波束指向 θ0 ，即让该方向的信号同相相加（输出模值最大），权矢量 W 应取为该方向的导向矢量：
W(θ0)=[1,e−jkdsin⁡θ0,...,e−jk(N−1)dsin⁡θ0]T

注意：实际计算中通常取共轭以抵消相位差，课件中展示了包含负号指数的形式，对应补偿相位。

ADBF

ADBF 的核心在于“自动调整”。它不仅仅是简单地合成波束，而是根据当前的电磁环境（信号和干扰的情况），智能地改变天线阵列的加权系数。

  • 自动调整方向图： ADBF 能够自动调整阵列的方向图形状 。
  • 抑制干扰与增强信号： 在复杂的信号环境中，ADBF 旨在自动抑制干扰（在干扰方向形成零陷/零点）并增强期望信号的能力 。
  • 旁瓣相消 (SLC)： 作为 ADBF 的一种典型应用结构，通过主天线与辅助天线的配合，在干扰方向自适应形成波束零点，从而对消干扰信号 。

    表中分别是最小均方误差准则 (MMSE)；最大信噪比准则 (MSNR)；线性约束最小方差准则 (LCMV)。

数字阵列雷达

数字阵列雷达的本质特征是“全数字化”。与传统相控阵雷达依靠模拟移相器来改变相位不同，数字阵列雷达通过数字方式直接控制信号。
三种主要的工作模式 ：

  1. SISO 模式（单输入单输出）： 收、发均为 DBF 的单波束模式。
  2. MISO 模式（多输入单输出）： 发射宽波束，接收采用 DBF 形成多波束。这种模式常用于需要快速覆盖大空域的场景。
  3. MIMO 模式（多输入多输出）： 在 MIMO 方式下工作，发射和接收都进行灵活的波束形成。
