问题1 低重频与高重频

一、 为什么低重频距离不模糊？（测距原理）

雷达测距的核心原理是测量回波的时间延迟 ( t )。

公式：

R=c⋅t2

1. 距离模糊的产生

雷达每隔一个固定的时间间隔发射一个脉冲，这个间隔叫 脉冲重复周期 ( Tr )，也就是脉冲重复频率 ( fr ) 的倒数： Tr=1/fr 。

如果在下一个脉冲发射之前，上一个脉冲的回波还没回来，雷达就会“搞错”。

  • 正常情况：脉冲1发射 -> 回波1回来 -> 脉冲2发射。雷达知道回波1属于脉冲1。

  • 模糊情况：脉冲1发射 -> 脉冲2发射 -> 回波1才回来。雷达会误以为这个回波是脉冲2产生的近距离回波，这就是距离模糊。

2. 最大不模糊距离公式 ( Rmax )

为了不产生模糊，回波必须在下一个脉冲发出前回来，即 t<Tr 。

代入测距公式：

Rmax=c⋅Tr2=c2fr

3. 为什么低重频 ( fr 低) 测距好？

  • 低重频 (Low PRF)： fr 很低（例如 1 kHz），意味着 Tr 很大（听得久）。

    • 推导： Tr 大 → Rmax 很大（例如 150 km）。

    • 结果：现实中的目标通常都在这个范围内，回波都能在周期内回来，不会发生重叠。所以距离不模糊，可以直接测距。

  • 高重频 (High PRF)： fr 很高（例如 200 kHz），意味着 Tr 极短。

    • 推导： Tr 小 → Rmax 很小（可能只有 750米）。

    • 结果：100 km 外的目标回波回来时，雷达可能已经发射了第100个脉冲了。雷达分不清这个回波属于哪一个脉冲，所以距离严重模糊。

二、 为什么高重频速度不模糊？（测速原理）

雷达测速的核心原理是测量多普勒频率 ( fd )。

公式：

fd=2vλ

( v 是径向速度， λ 是波长)

1. 速度模糊的产生 (奈奎斯特采样定理)

雷达发射脉冲串相当于对目标的多普勒信号进行离散采样。采样率就是 重频 ( fr )。

根据信号处理的奈奎斯特采样定理，为了不发生混叠（Aliasing），采样率必须大于信号频率的2倍。

即：

|fd|<fr2

如果目标速度太快，产生的多普勒频率 fd 超过了 fr/2 ，雷达测出的速度就会出错（发生折叠），这就是速度模糊（或者叫盲速）。

2. 最大不模糊速度公式 ( vmax )

将 fd=fr/2 代入多普勒公式：

2vmaxλ=fr2

解出：

vmax=λ⋅fr4

3. 为什么高重频 ( fr 高) 测速好？

  • 高重频 (High PRF)： fr 很高（例如 200 kHz）。

    • 推导： fr 大 → vmax 很大（能覆盖几马赫的速度）。

    • 结果：战斗机、导弹等高速目标产生的多普勒频移都在这个范围内，不会发生混叠。所以速度不模糊。

  • 低重频 (Low PRF)： fr 很低（例如 1 kHz）。

    • 推导： fr 小 → vmax 很小（可能只有几十米/秒）。

    • 结果：稍微快一点的飞机，其多普勒频率就会超过 fr/2 ，导致测速出现严重的“虚假值”（比如把接近看成远离，或者速度数值完全错误）。所以速度模糊。

三、 总结与对比表

这就形成了一个“按下葫芦浮起瓢”的局面，可以通过下表直观对比：

补充说明（工程解决）：

为了解决这个矛盾，现代雷达通常采用 中重频 (Medium PRF) 或者 多重频参差 (Stagger PRF) 技术：

  • 在不同的时间段快速切换不同的 PRF，利用算法（如中国余数定理）来解算出真实距离和真实速度，从而兼顾两者的优点。
